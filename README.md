# tommo's first cookie-cut cookie-cutter

To use:

```zsh
$ cookiecutter path/to/.cookie
$ cd [project-slug]
$ profit!
```

Be nice and:

* add your modules to the `pyproject.toml` import statement
* import your submodules in `__init__.py`
